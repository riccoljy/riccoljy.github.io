Projects

The A-maze-ing Race | Maze-solving Robot

The "The A-maze-ing Race" project involved a maze which required us to create a robot which can solve through the maze in the shortest amount of time.

Along the maze, there are various pieces of coloured paper on the floor which our robot needs to detect and react differently in order to complete the race. The colours help determine which direction the robot has to move in through the maze at a particular way point.

Obstacle Course Robot

The objective of this project is to create a robot that is powered by a battery and is capable of navigating through an obstacle course. The robot must then stop 5 cm away from a wall and deliver a ping pong ball over it. The robot must fit within a cube of 30cm x 30cm x 30cm, and can only be constructed using the provided materials.

The time limit for completing the course is 60 seconds.
